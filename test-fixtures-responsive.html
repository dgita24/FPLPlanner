✅ What to Check

  • Desktop (>767px): Look at the player cards - you should see the team abbreviation (e.g., "ARS", "LIV") and ALL THREE future fixtures below the card
  • Mobile (≤767px): The team abbreviation should disappear, and only TWO fixtures should be visible (the middle one is hidden)
  • Touch Targets: On mobile, all buttons should be easily tappable (48x48px minimum)
  • Layout: Cards should adjust size appropriately at each breakpoint

🔍 CSS Rules Applied

@media (max-width: 767px) {
  /* Hide the 3rd fixture (middle child in future-fixtures) */
  .future-fixtures .fixture:nth-child(2) {
    display: none;
  }
  
  /* Hide team abbreviation for better spacing */
  .player-card .info .team {
    display: none;
  }
  
  /* Apply utility classes */
  .hide-mobile,
  .desktop-only {
    display: none !important;
  }
}
