🚩 Keane Suspension Fix - Date-Based Parsing

The Problem

Original Issue: Keane shows as red flagged for the whole season even though his suspension ends on Jan 31.

Root Cause: The system couldn't parse date-based suspension news like "Suspended until 31 January".

The Fix

❌ Before

  • Only parsed "Suspended for X matches"
  • Couldn't handle date-based news
  • Keane flagged indefinitely

✅ After

  • Parses dates like "31 January"
  • Converts dates to gameweeks
  • Flag removed after GW23

Keane's Case: "Suspended until 31 January"

Current Gameweek: 22

Suspension Until: January 31, 2026

Logic: Player is unavailable until end of Jan 31. GW23 deadline is Jan 31 at 14:00, so he misses GW23. Available from GW24 onwards.

Implementation Details

Step 1: Parse the Date

news: "Suspended until 31 January" ↓ Regex match: /(?:until|back|return)\s+(?:on\s+)?(\d+\s+January)/i ↓ Parsed: Date(2026, 0, 31) // Jan 31, 2026

Step 2: Find Last Suspended Gameweek

Compare suspension date with gameweek deadlines: - GW22 deadline (Jan 24) <= Jan 31? YES ✓ Suspended - GW23 deadline (Jan 31 14:00) <= Jan 31 23:59? YES ✓ Suspended - GW24 deadline (Feb 7) <= Jan 31? NO ✗ Available ↓ Last suspended GW: 23

Step 3: Show/Hide Flag Based on Viewing GW

if (viewingGW > suspensionEndGW) { // Don't show flag return false; } // Otherwise show flag Examples: - Viewing GW22: 22 > 23? NO → Show flag ✓ - Viewing GW23: 23 > 23? NO → Show flag ✓ - Viewing GW24: 24 > 23? YES → Hide flag ✓ - Viewing GW25: 25 > 23? YES → Hide flag ✓

Supported Date Formats

  • ✅ "Suspended until 31 January" (Keane's format)
  • ✅ "Suspended until Jan 31"
  • ✅ "Expected back 7 February"
  • ✅ "Suspended until February 14th" (with ordinal suffix)
  • ✅ "Suspended for 2 matches" (original format still works)

Additional Enhancements

chance_of_playing Fields

For current and next gameweek, the system now also checks:

  • chance_of_playing_this_round - If 100%, no flag shown
  • chance_of_playing_next_round - If 100%, no flag shown

This provides additional accuracy beyond just parsing the news text.

Keane now correctly shows:

  • 🟥 Red flag in GW23

The system now handles date-based suspensions accurately!
